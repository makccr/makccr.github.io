Downloads

Here you can download all of the projects I've made available over the years. Themes, After Effects presets, wallpapers, and much more. If you have any questions, comments, or suggestions about anything on this page, use my contact page.

Fonts

P.R. Onions

A font face that looks like it is handwritten, only because it was quite literally written by hand and then scanned into a font.

Scripts & Configs

Dot Files

Configs for WMs, Vim, Polybar, & a lot more. There are some additional niche scripts. My default storage spot for ricing.

Suckless Apps

A git repo containing my personal builds for all of the suckless apps that I use. They're super light-weight, but look dope.

Disk Monitor

A polybar module based on a simple bash script using df, grep, & awk that displays the amount of space used on a root linux drive.

YT Polybar

A polybar module, powered by a python script originally forked from HowCode.org, that displays the subscriber count for a channel.

AWMP

A polybar module that I built to display all nine Awesome WM tags, and enable tag switching via mouse clicks & keyboard shortcuts.

Minecraft Sync

A bash script I set-up to copy Minecraft save games to Dropbox, and then create symbolic links to create a cloud saved game.

Wallpapers

Wallpapers

My main collection of wallpapers. I didn't make any of these, I've just curated some nice, high-res wallpapers for people to use.

Paper

A very large collection of retro, neon, style wallpapers featuring the logo for just about any linux distro you could want.

Retro Tech

A collection of retro, sort of 8-bit wallpapers, based on the logos for old technology companies. I got really bored at 2AM.

Minimal Vim

A neumorphism sytle wallpaper I created, based on the logo for Vim, my favorite text editor. Light and dark version included.

Minimal Yin

Another Neumorphism style wallpaper deisgn, based on the Yin and Yang symbol. Light and dark version included.

AE Presets

Shoot on Film

This is a collection of After Effects presets I created to try and emulate the film look gaind from shooting with different film stocks.

CC Preset

A preset for Adobe After Effects that acts as the base for simple grading jobs. I know most people don't grade in AE, whatever...

App Themes

Alfred Themes

A set of nice themes that I created to work with the Spotlight replacement, Alfred. Themes are designed to be light-weight.

Ulysses Themes

Two themes that I created for Ulysses, my favorite GUI markdown editing app. I prefer Vim generally, but this is nice too.
